【Home.tsx】
- 問候語依時段（早/午/晚/夜）從 GREETING_MESSAGES 隨機取
- XP：daysToNext = ceil((next.minXp - xp) / 12)，顯示「N 經驗值 · 還需 N 天升階」
- 今日狀態（SpecialStateType）：menstrual/cold/fatigue/poor_sleep，位於 DailyCard 上方
- 本週日曆：7 天 dot 視圖，今天有外框
- 已移除：今日提示 Card、首次記錄空白狀態 Card

【DailyCard.tsx】(src/components/ui/DailyCard.tsx)
- CARD_H：未翻 172px / 翻牌後 240px
- 背面（未翻）：BrandLogo(size=72) + "今日意圖牌" + 日期 + 動畫 tap hint
- 狀態選擇器在卡片「下方」（非卡片內），未翻時顯示
- 翻牌後正面顯示：勵志文字 + 分隔線 + "今日行動" 任務（無狀態標籤、無 CTA 按鈕）
- 牌卡資料：getCard(bodyIndex, mindIndex, variant) → {text, task}
- 25 組合 × 3 variant = 75 張
- localStorage key：'yoga_bloom_daily_card'，含 {date, bodyIndex, mindIndex, flipped, variant}

【BrandLogo.tsx】(src/components/ui/BrandLogo.tsx)
- 自訂 SVG 冥想人像
- Props: { size?: number }，預設 120
- 漸層身體（sage→sandy），三個脈輪圓點（粉#EDB0B8, 藍綠#8ABFBF, 米#ECEADE）
- 胸前白色符號（open book + stem + petal）

【BodyPartSelector.tsx】(src/components/ui/BodyPartSelector.tsx)
- SVG viewBox="0 0 120 185"，size 120×185
- 人體圖：較細長版，torso 寬度約 60px（原本 76px）
- REGIONS 10 個可點擊區域（circle + rect），click = toggle 選取
- REGION_MAP：{calves2: 'calves'}（左右小腿 merge 為同一部位）
- 右側：BODY_PARTS 藥丸按鈕（點擊展開動作建議）
- 動作建議卡片：無 emoji，顯示 pose.name + pose.nameEn + pose.description

【PosePicker.tsx】(src/components/ui/PosePicker.tsx)
- 分類篩選列：style={{ overflowX: 'auto', WebkitOverflowScrolling: 'touch', touchAction: 'pan-x' }}
  （flex-shrink-0 在每個 button，確保橫向滑動）
- 3 欄圖片網格，tap 選取 + tap 再次打開詳情面板
- 詳情面板：左側圖片 + 右側名稱/難度/操作按鈕
- 可新增自訂動作

【Partners.tsx】(src/pages/Partners.tsx)
- 11 個商家（3 免費 + 8 付費/Premium）
- 分類：全部/商店/課程/活動/商品
- PartnerCard：<button> 容器，點擊開啟 PartnerDetailSheet
- PartnerDetailSheet：fixed bottom-0，AnimatePresence，spring 動畫
  - 外層：className 無 overflow-hidden（重要！否則內容無法滑動）
  - 內層：overflow-y-auto，maxHeight: calc(88vh - 24px)
  - 內容：fullDescription / 地址時段網站 / 亮點 / 如何兌換 / CTA 按鈕
